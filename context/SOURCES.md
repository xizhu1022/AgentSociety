## How to add sources

1. Drop files into `context/papers/`, `context/notes/`, or `context/prior_work/`
2. Run `/research:context` to read and index them
3. Or: add a URL row to the table above — the AI will fetch it during PLAN

## Source types

- **paper** — PDF or text from academic papers
- **notes** — Your research notes, ideas, advisor feedback
- **prior_work** — Existing code, baselines, prior results
- **url** — Web URL the AI should fetch and read
- **data** — Dataset descriptions, baseline metrics
